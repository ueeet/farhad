ФОТО ДЛЯ САЙТА
==============

Сейчас на сайте используются плейсхолдеры (тёмные карточки с подписями).
Чтобы заменить их на реальные фото, положите файлы в эту папку:

  /public/images/

И затем замените код в компонентах:

1. ГЛАВНАЯ ФОТО (Hero)
   Файл: src/components/Hero.tsx
   Найдите блок с классом "hero-photo" и замените содержимое на:
     <img src="/images/hero.jpg" alt="Фархад Иванов" className="absolute inset-0 w-full h-full object-cover" />

2. ГАЛЕРЕЯ
   Файл: src/components/Gallery.tsx
   В массиве items добавьте поле image к каждому объекту:
     { tag: "Wedding", title: "...", h: "tall", image: "/images/01.jpg" }
   А внутри карточки вместо плейсхолдера поставьте:
     <img src={it.image} alt={it.title} className="absolute inset-0 w-full h-full object-cover" />

Рекомендуемые названия:
  hero.jpg            — главный портрет (вертикальный, минимум 800x1066)
  01.jpg ... 08.jpg   — для галереи (минимум 800x800 для квадратных, 800x1066 для вертикальных)

Формат: JPG (предпочтительно) или WebP.
Размер каждого файла — не больше 500 КБ для скорости.
